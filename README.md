## 💎 Example: Simplicity Incarnate

```rust
use layer9::prelude::*;

#[component]
fn Counter() -> Element {
    let count = use_state(|| 0);
    
    view! {
        <Card>
            <h2>"Count: "{count.get()}</h2>
            <Button on_click={move |_| count.set(count.get() + 1)}>
                "Increment"
            </Button>
        </Card>
    }
}

// That's it. No useEffect. No hydration errors. No tears.
```
